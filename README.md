# DeepSeek Desktop

Unofficial desktop client for [DeepSeek Chat](https://chat.deepseek.com), built with Electron.

## Features

- Native desktop wrapper for DeepSeek Chat
- Persistent login session
- System tray integration
- Global shortcuts (toggle window, new/delete conversation)
- Desktop notifications for new messages
- Local conversation cache for faster loading
- macOS & Windows support

## Shortcuts

| Action | macOS | Windows |
|--------|-------|---------|
| Toggle window | Cmd+Shift+D | Ctrl+Shift+D |
| New conversation | Cmd+Shift+N | Ctrl+Shift+N |
| Delete conversation | Cmd+Shift+Backspace | Ctrl+Shift+Backspace |

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run dist
```
